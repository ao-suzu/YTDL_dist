# YouTube Downloader 導入ガイド

この度は「YouTube Downloader」をダウンロードいただき、ありがとうございます。
本ツールを快適にお使いいただくためのセットアップ手順をご案内します。

準備は大きく分けて2つのステップ（専用プログラムの配置とブラウザへの登録）で完了します。

---

## 1. 専用プログラムのセットアップ（初回のみ）

この拡張機能は、動画を変換するために専用のプログラム（Native Host）を利用します。
別途 Python などの環境を構築する必要はありません。以下の手順でセットアップを行ってください。

1.  **`host` フォルダを開く**
    本パッケージに含まれている `host` フォルダをエクスプローラーで開きます。
2.  **`install.bat` を実行する**
    フォルダ内の `install.bat` をダブルクリックして実行してください。
    > [!NOTE]
    > **「WindowsによってPCが保護されました」という画面が表示された場合**
    > 「詳細情報」をクリックし、右下に表示される「実行」ボタンを押してください。
3.  **画面の指示に従い、IDを入力する**
    黒い画面（コマンドプロンプト）が表示されます。途中で「Extension ID」の入力を求められます。
    IDの確認方法は、次の「ステップ2」で詳しく説明します。

---

## 2. ブラウザへの拡張機能登録

次に、Google Chrome に拡張機能を登録し、IDを取得します。

1.  **拡張機能管理画面を開く**
    Chrome を起動し、アドレスバーに `chrome://extensions/` と入力して移動します。
2.  **デベロッパーモードを有効にする**
    画面右上の **「デベロッパーモード」** のスイッチを **ON** にします。
3.  **フォルダを登録する**
    本パッケージの **`extension` フォルダ**を、そのままブラウザの管理画面上に **ドラッグ＆ドロップ** してください。
4.  **IDをコピーして、セットアップを完了させる**
    登録された拡張機能のカード内に表示されている **「ID」**（英数字の羅列）をコピーします。
    先ほど（ステップ1）で開いた黒い画面に、このIDを貼り付けて Enter キーを押せば、すべての設定が完了します。

---

## 使いかた

1.  YouTube の動画ページ、または再生リストのページを開きます。
2.  ブラウザ右上の「拡張機能（パズル玉のアイコン）」から、「YT Downloader」を探してピン留めします。
3.  アイコンをクリックするとサイドバーやポップアップが表示されるので、**「ダウンロード」** ボタンをクリックしてください。

---

## お困りのときは

- **「セットアップが必要です」と表示される場合**
  ステップ1の `install.bat` による設定が正しく完了しているか、入力した ID が間違っていないかを確認してください。
- **ダウンロードが開始されない場合**
  設定を反映させるため、一度ブラウザを完全に閉じてから再起動することをお勧めします。
